# mirror-quad-parts

Supported on MuseScore 4.5

## How to Install
1. Put mirror-quad-parts.qml in Documents/MuseScore4/Plugins

## Recommendation
Add shortcut via Preferences > Shortcuts > Run plugin mirror-quad-parts
_Recommended shortcut: CMD Option M_
